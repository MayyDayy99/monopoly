# Terminológia Szótár

A játékban a klasszikus Monopoly fogalmakat a Loricatus Group technológiájára kell cserélni.

| Klasszikus Monopoly | Loricatus Technológia |
| :--- | :--- |
| Ház (House) | **LOD Szint** |
| Hotel | **Digitális Iker** |
| Börtön | **Repülési Tilalmi Zóna** |
| Esély (Chance) | **Terepi Munka** |
| Közösségi Alap | **Adatfeldolgozás** |
| Adó | **Licencdíj** |
| Jelzálog | **Szolgáltatás szüneteltetése** |

## Ingatlanok és Mezők
- **Utcák**: Magyar műemlékek (pl. Zsámbéki Romtemplom, Budavári Palota).
- **Állomások**: Tech-eszközök (pl. 3D Lézerszkenner, RTK Bemérés).
- **Közművek**: Adatszolgáltatások (pl. Felhő alapú tárhely, GIS adatbázis).
